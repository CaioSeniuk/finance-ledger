## Description

A finance ledger developed in NestJS, TypeScript, Kafka, GraphQL and microservices architecture.


Dependencies version:
npm -v
- 11.12.1

nvm -v
- 1.2.2

node version
- 24.15.0


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
